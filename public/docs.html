Beats.IO

(v0.0.1)

WARNING: This software is not yet production-quality. In fact, it's pretty much not any quality at all yet. Work is in progress, but use at your own risk.

Beats.IO is a CoffeeScript live coding environment designed to make it easy to make sounds and music from inside a web browser. It works on recent builds of Chrome and Firefox and is built on a few awesome frameworks and libraries.

To run the code you've written, try Cmd-S (mac) or Ctrl-S (win/lin). Audiolet.js and Music.js are provided. Music.js can be used directly, and Audiolet is pre-initialized as @audio. In addition, some convenience functions are provided to make live coding easier and faster. The killSound function is particularly useful when things get out of hand.

API

bcon(node)
Adds a node to @audio.output.
bdis(node)
Remove a node from @audio.output.
bfreq(note)
Returns a frequency, given a Latin-style note (i.e. bfreq('C4') === 261.6255653005986)
bsine(noteOrFreq)
Returns sine wave generator, given a frequency or note. Use it with bcon to make sound!
bsaw(noteOrFreq)
Like bsine except for saw waves.
killSound()
Removes all nodes from output, thus killing all the sounds.
